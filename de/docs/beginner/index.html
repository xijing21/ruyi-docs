RuyiSDK 新手入门指南

1. RuyiSDK 简介​

RuyiSDK 是中国科学院软件研究所开发的一个开源套件，它是一个面向 RISC-V 开发的工具套件。更多产品详情参考RuyiSDK简介。

本文以 Milk-V Duo RISC-V 开发板为例，介绍如何从 RuyiSDK 开始使用设备。

2. 从 ruyi 开始​

2.1 安装 RuyiSDK 包管理器​

  1. 下载安装：

    • 下载地址：https://mirror.iscas.ac.cn/ruyisdk/ruyi/releases/
    • 下载最新版本，参考安装ruyi章节
    • 安装后运行 ruyi -V 检查版本

  2. 初始设置：

    # 更新存储库索引
    ruyi update

    # (可选)切换为备用存储库（解决GitHub网络问题）
    ruyi config set repo.remote "https://mirror.iscas.ac.cn/git/ruyisdk/packages-index.git"

    # 设置遥测（建议开启）
    ruyi telemetry consent

    提示：建议定期运行 ruyi update 保持工具最新，执行 ruyi news 获取最新动态。

2.2 基本使用​

请使用 ruyi -h 命令 以及 ruyi <subcommands> -h 命令查看帮助信息。最基础和最常用的命令如下所示，更多 ruyi使用，详见功能概述章节。

# 列出可用软件包
ruyi list --name-contains <name-str>
ruyi list --category-is <category>
# ruyi list --name-contains ''
# ruyi list --category-is 'toolchain'
# ruyi list --name-contains 'plct'

# 安装开发工具（如工具链、模拟器）
ruyi install <package-name>

# 下载解压demo
ruyi extract <package-name>

# 创建虚拟环境
ruyi venv 

# RISC-V设备系统安装引导
ruyi device provision

3. 开发板系统安装​

为RISC-V开发板安装系统您需要使用如下命令：

# RISC-V设备系统安装引导
ruyi device provision

参考 为 MilkV Duo 开发板刷写镜像，了解详细的镜像安装过程。相关视频供参考。

4. 编译程序并运行​

执行编译，需要先安装编译工具链，请参考 为 MilkV Duo 开发板编译riscv程序并运行验证，了解详细的编译准备和编译运行过程。相关视频供参考。

5. 更多 RISC-V 开发板支持​

可前往 support-matrix项目 查询了解。访问 support-matrix项目Web前端 查看更友好。
