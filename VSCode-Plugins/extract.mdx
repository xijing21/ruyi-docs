# 源码包提取（Extract RuyiSDK Package）

在 VS Code 资源管理器中右键目标文件夹，选择 **Extract RuyiSDK Package**，即可直接下载并解包官方示例或源码包。

![alt text](./image/image-7.png)

## 典型流程

1. 准备一个工作目录，右键选择 **Extract RuyiSDK Package**。
2. 选择要提取的包（例如 `coremark`、`ruyisdk-demo` 等）。
3. 解包完成后按项目自述文件进行编译/运行。

## 提示

- 建议先在 **Packages** 中安装工具链，再创建并激活虚拟环境后进行构建。
- 构建示例可参考[CoreMark（以 LicheePi 4A 为例）](./cases/case1.md)。
- 如需镜像/刷机，请参考Ruyi 包管理器文档中的相关案例，插件暂未提供刷机向导。
